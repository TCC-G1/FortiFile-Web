Bienvenido a la sección de ayuda

Aquí encontrarás instrucciones y consejos para aprovechar al máximo FortiFile.

1. Instalación

  1. Descarga la versión más reciente desde la sección de Descargas.
  2. Ejecuta el instalador y sigue las instrucciones en pantalla.
  3. Abre la aplicación desde el acceso directo o menú de inicio.

2. Inicio de sesión

  • Abre FortiFile.
  • Introduce tu usuario y contraseña.
  • Si es tu primer inicio, registra una nueva cuenta.

3. Funciones principales

  • Subir y organizar archivos de manera segura.
  • Crear carpetas y mover archivos entre ellas.
  • Buscar archivos por nombre o etiquetas.
  • Configurar opciones de seguridad y backups.

4. Consejos

  • Realiza backups regularmente.
  • Utiliza nombres descriptivos para tus archivos.
  • Consulta la sección “Acerca de” para información sobre actualizaciones.
